# Dashboard

The Dashboard provides a high-level overview of the entire AITradingSystem system state.

It serves as the main entry point to see what's running, what's failing, and what actions have been taken recently.

---

## Key Metrics

The top of the dashboard displays key metrics:
- **Active Triggers**: Number of triggers currently online.
- **Active Groups**: Number of trading groups configured.
- **Open Positions**: Total number of currently open trading positions.
- **Unrealized PnL**: Total unrealized profit/loss across all open positions.

---

## Recent Activity

A feed of the most recent events and system messages, giving you a real-time view of what's happening.

This includes:
- Signals received from triggers.
- Decisions made by trading groups.
- Order execution status.
- System errors or warnings.
